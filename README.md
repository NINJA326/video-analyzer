# NINJA AIRS VIDEO ANALYZER

GitHub Pages公開用の更新版です。

## アップロード方法

このZIPを展開し、中のファイルとフォルダを `video-analyzer` リポジトリのルートへアップロードしてください。

構成:

- `index.html`
- `css/style.css`
- `js/*.js`

GitHub Pages:
- Source: Deploy from a branch
- Branch: main
- Folder: / (root)

公開URL:
`https://ninja326.github.io/video-analyzer/`

## 注意

- MP4モード: 録画・画像保存・コマ送り対応
- YouTubeモード: 再生・停止・速度変更・書き込み・コメント・場面保存対応
- YouTube映像を含む録画・画像保存はブラウザ制限により非対応
- Chrome推奨


## YouTube解説録画

1. YouTube URLを読み込みます。
2. `YouTube解説録画`を押します。
3. Chromeの共有画面で「このタブ」を選択します。
4. 「タブの音声も共有」をONにします。
5. 動画を再生し、ペン・矢印・コメントで解説します。
6. `録画終了`を押すと `ninja_youtube_commentary.webm` が保存されます。
7. 保存動画をYouTube Studioからアップロードします。

### 注意
- 自分が権利を持つ動画、または利用許諾を得た動画のみ使用してください。
- YouTube映像を直接ダウンロードする機能ではありません。
- ブラウザの画面共有機能を使って解説画面を録画します。
