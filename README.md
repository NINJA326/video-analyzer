# NINJA AIRS VIDEO ANALYZER MP4 v19.2.1

MP4特化の動画分析・解説アプリです。

## 設計方針

ClipchampやCanvaなど一般的な動画編集画面を参考に、次の優先順位で再設計しています。

1. 動画プレビューを最大化
2. シークバーを動画直下に配置
3. すべての操作を1か所のコントロールバーに集約
4. 指導コメントと場面一覧を右側へ分離
5. YouTube関連機能を完全に削除して安定性を向上

## 主な機能

- MP4読込・ドラッグ&ドロップ
- 再生・停止・±5秒
- 正確なコマ送り
- 0.25～2倍速
- ペン・直線・曲線矢印・円・四角
- 元に戻す・やり直し・全消去
- コメント・音声入力
- コメント画面表示
- 選手選択
- 場面保存・移動・削除
- 画像保存
- 高画質録画
- 作業データの途中保存・再開

## AI自動配置について

現在フレームから選手10人とボールを正確に配置するには、
専用の選手・ボール検出モデル、チーム識別、コート座標変換が必要です。
GitHub Pagesだけで簡易検出を入れると誤配置が多くなるため、
v19.2.1では安全のため準備中表示にしています。
